Flappy Bird

Hallo und Wilkommen zu meiner Version von Flappy Bird :)

Wie im Namen schon gesehn, hab ich ein Flappy Bird Programmiert, dieses basiert auf dem alt bekannten Flappy Bird.
Im Rahemn der LB2 vom Informatik Modul 226b, mussten wir ein kleinen Spiel Programmieren und so unsere Java Kenntnise beweisen.
Mit hilfe von Greenfoot wurde das Spiel erstellt und getestet.



 

---------------------------------------------------------------------------------------------------------------------------------------------------
Wie spiele ich das Spiel?

Um das in den Genuss des Spieles zu kommen, müssen Sie als erstet Greenfoot auf ihrem PC installieren. Danach können Sie den Ordner mit den Dateien darin öffnen.

---------------------------------------------------------------------------------------------------------------------------------------------------------------
Infos zum Spiel

Das Spiel wird mit dem grünen Play-Button gestartet. Mit der Leertaste fliegt unser Vogel, wie bei einem ganz normalen Flappy Bird
Das Spiel startet im Level 1, diese Level geht bis 25 Punkte, danah startet automatisch das Level 2.


Viel Spass!
